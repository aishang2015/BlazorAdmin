### BlazorAdmin

一个使用 Blazor 和 MudBlazor 编写的简单管理项目模板。

---

#### 登录凭据：

用户名：BlazorAdmin

密码：BlazorAdmin

---

#### 特性：

- 支持交互式自动渲染模式

- 实现了基于角色的访问控制（RBAC）

- JWT 身份认证

- 基于 MudBlazor 构建

- 主题明暗及颜色切换

- 本地化支持

- 审计/登录日志

- 模块化设计

- 添加Quartz支持

---

#### 技术栈：

- .NET 9

- Blazor

- MudBlazor

- Entity Framework Core
