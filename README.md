A simple management project template written using Blazor and MudBlazor.

#### Features:

- Support Interactive Auto Render Mode

- Implemented RBAC

- JWT Authentication

- Built on Mudblazor

- Theme light/dark and color switch

- Localization

- Audit/Login Log

- Modularization

- Quartz Support

#### Technology stack:

- .NET 9

- Blazor

- MudBlazor

- Entity Framework Core
